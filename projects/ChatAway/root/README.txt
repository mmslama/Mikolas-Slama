Mikolas Slama - Final Project - ChatAway

To start the server simply type "npm start" this will install all the dependencies and run the server. 

Open the webbrowser and go to localhost:3000 which will direct you to the login page. If you do not have an account hit register and
create an account.

Once you have an account you will be redirected back to login. Once loged in, simply type the room you would like to go into and hit
the enter button. Chat as you please and hit logout once you wish to end the experience.
